The previosly buggy Memory Game:

I copied the code from a YouTubers' course but noticed a litle bug. The function that pics and flips a card was still available to be called while the evaluation for a match was made So if a user clicks a 3rd card before the evaluation completes the array resets while the 3rd cards' image source is not reset. So I decided to completely remove the eventListener's while evaluating, it prooved to be a challenge as the node lists' eventlistener's needed to be amended again after a match was cheked for. I then used the ES6 forEach array method to loop thru the node list accomponanied with a new array I created of correct card's chosen to re add the eventlistener's after the evaluation was compleet.
